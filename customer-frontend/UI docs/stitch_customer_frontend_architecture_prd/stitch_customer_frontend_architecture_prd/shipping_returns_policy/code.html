Returns Policy

We offer a 30-day return window for all items in their original condition. We believe in our craftsmanship, but we understand that finding the perfect fit is a personal journey.

Conditions for Return:

  • check_circle Items must be unworn, unwashed, and with all original tags attached.
  • check_circle Footwear must be tried on carpeted surfaces and returned in the original box.
  • check_circle Skincare and fragrance items must remain sealed in original packaging.

Initiate a Return
