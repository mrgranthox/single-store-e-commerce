The Narrative

The Sculpted Trench is an exercise in restrained luxury. We've removed the traditional military epaulettes and excessive hardware to focus on the purity of the silhouette. Every line has been considered to create an architectural profile that moves with the body.

Fabricated from our signature double-faced wool, this piece offers exceptional warmth without the bulk. The interior is finished with hand-rolled silk seams, a hallmark of atelier craftsmanship usually reserved for haute couture.

Craftsmanship Details

  • Hand-finished blind hems
