"Incredibly lightweight and reactive"

I've been using the Swift-Core Runner Pro for about two weeks now. As a marathon trainer, I'm very picky about responsiveness. These shoes deliver exactly what they promise. The transition from heel to toe is seamless. After 50 miles, the cushioning still feels as fresh as day one. Only downside is the laces are a bit short, but that's an easy fix. Highly recommend for serious runners.
